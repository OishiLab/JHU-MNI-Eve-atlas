
# JHU-MNI atlas (Eve atlas)


The JHU-MNI (Eve) atlas is a single-subject brain atlas that includes a structural parcellation map developed at Johns Hopkins University and distributed in Montreal Neurological Institute (MNI) standard space. These atlases are widely used in neuroimaging for anatomical labeling, region-of-interest (ROI) analysis, and cross-subject normalization, particularly in diffusion MRI studies.<br>
